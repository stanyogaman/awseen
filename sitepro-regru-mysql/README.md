# сайт-про.рф — PHP/MySQL версия для обычного хостинга REG.RU

Лёгкая версия сайта для загрузки на обычный PHP/MySQL хостинг.

## Установка

1. Создайте MySQL базу в панели REG.RU.
2. Загрузите файлы в папку сайта, например `public_html`.
3. Откройте `https://сайт-про.рф/install.php`.
4. Введите MySQL данные и создайте администратора.
5. После установки удалите `install.php`.
6. Вход в админку: `/admin/login.php`.

## Возможности

- Главная страница на русском в стиле AWSEEN / сайт-про.рф
- MySQL база данных
- Простая админка
- Настройки сайта: телефон, email, Telegram, WhatsApp, VK, адрес, скрипты
- Пакеты услуг и цены
- Страницы и SEO-поля
- Заявки сохраняются в MySQL
- n8n webhook для Google Sheets / Яндекс Таблиц / CRM
